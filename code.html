The Code for Fizz Buzz

                        
                            function getValues() {
                                //get user values from the page
                                let fizzValue = document.getElementById('fizzValue').value;
                            
                                let buzzValue = document.getElementById('buzzValue').value;
                            
                            
                            
                                //check to make sure they're numbers - parse
                            
                                fizzValue = parseInt(fizzValue);
                                buzzValue = parseInt(buzzValue);
                            
                            
                                //check to make sure they're integers
                            
                                if (Number.isInteger(fizzValue) && Number.isInteger(buzzValue)) {
                                    //call logic function -call fizzBuzz
                                    let fbArray = fizzBuzzC(fizzValue, buzzValue);
                                    //call displayData and write the values to the screen
                                    displayDataB(fbArray);
                                } else {
                                    Swal.fire(
                                        {
                                            icon: 'error',
                                            title: 'Oops!',
                                            text: 'Only integers are allowed for Hundo!'
                                        }
                                    );
                                }
                            
                            }
                            function fizzBuzzC(fizzValue, buzzValue) {
                                let returnArray = [];
                            
                                for (let i = 1; i <= 100; i++) {
                                    let value = ((i % fizzValue == 0 ? 'Fizz' : '') + (i % buzzValue == 0 ? 'Buzz' : '') || i);
                                    returnArray.push(value);
                                }
                                return returnArray;
                            }
                            function displayDataB(fbArray) {
                                let resultsDiv = document.getElementById('results');
                            
                                let resultsHtml = '';
                            
                                for (let i = 0; i < fbArray.length; i++) {
                                    let returnArray = fbArray[i];
                                    let className = '';
                            
                                    if (returnArray == 'FizzBuzz') {
                                        className = 'FizzBuzz';
                                    } else if (returnArray == 'Buzz') {
                                        className = 'Buzz';
                                    } else if (returnArray == 'Fizz') {
                                        className = 'Fizz';
                                    }
                            
                                    resultsHtml += `
${returnArray}
`;
                            
                                }
                                resultsDiv.innerHTML = resultsHtml;
                            }
The code is structured in three functions although, there are several ways to solve this challenge - the code displayed here is the version that's currently running

Please Note: Some of the characters may not display correctly, see the repo (linked in navigation menu) for full details.

function getValues()

is the controller function. The program begins by getting the user values from the page via getElementById. It then checks to make sure that the input values are integers and displays a sweet alert if there is a user error. Also, we call the logic and display functions here.

function fizzBuzzC

is the logic function. This function begins by declarring returnArray as an empty array. Next, a for-loop is used to loop through the start and end values from the user's input. Modulo operators are also used here to determine if the user's input values are divisible by either 'fizz' value, 'buzz' value or BOTH ('fizz buzz'). Therefore, the app will return an array of numbers, but for those divisible by the startValue, endValue, or both will return the words 'FIZZ / BUZZ' respectively.

function displayDataB

is the view function. First, it grabs the 'results' id from the document in order to display in the proper place. Next, resultsHTML is declared as an empty string so that it is clear/empty for the user to begin using the app. Then, a for-loop loops through the numbers and then an if/else statement is used to add classNames to the respective values so that it displays WORDS instead of the numbers in the user's view on the app if the numbers are divisble by fizz, buzz, or both. If the numbers aren't divisible by those, then it will display the normal integers.
